Framework

-Engine
00 Base
Base : 레퍼런스 카운트 기능을 강제적으로 쓰기위한 상속 부모

01 Defines
Engine_Defines : 각종 헤더, 메모리누수 체크 매크로
Engine_Enum : Engine에서 사용할 Enum 모음
Engine_Function : 메모리 할당, 제거를 위한 안전함수 모음Safe~ 
Engine_Macro : 코딩 편의성을 위한 매크로 모음
Engine_Struct : Engine Struct 모음
Engine_Typedef : 각종 자료형을 임의의 이름으로 변환해놓은것
NoAssimpModelStruct : Assimp 라이브러리 의존성 제거를 위해 필요한 데이터 구조체 모음

02 System
Graphic : Device(데이터 모음), Context(기능 모음)을 만들고 출력을 위한 세팅함
Input : DInput을 이용한 초기화, 상태값 조회

03 Utility
EngineUtility : 외부에서 사용할 유틸리티 함수를 모아놓은 싱글턴 클래스 
FontManager : 폰트를 등록, 그리는 기능
IMGUIManager : IMGUI를 사용하기위한 패널을 등록, 그리는 기능
LightManager : 빛 데이터를 등록, 참조하는 기능
ObjectManager : 사본 객체들을 보관
Pipeline : 랜더링을 위한 행렬 데이터 모음
PrototypeManager : 원형 객체들을 보관
RenderManager : 화면에 출력하기위한 객체들을 보관, 출력
SceneManager : 씬 변경과 그에 따른 데이터 정리를 관리
TimeManager : 게임에 사용할 타이머 등록, 업데이트, 시간 참조
